## Run
### 以[BubbleSort.cpp](./sorting/BubbleSort.cpp)文件为例运行步骤如下：
1. 进入对应目录 `cd sorting`
2. 编译cpp文件 `g++ BubbleSort.cpp ./lib/DoubleVector.cp` ，默认会输出 `a.out` 文件 (具体命令视情况而定)
3. 执行编译后的文件 `./a.out`

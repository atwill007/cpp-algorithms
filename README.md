## Run
> 以[sorting/QuickSort.cpp](./sorting/QuickSort.cpp)文件为例运行步骤如下：

### Normal Run
- Enter target directory `cd sorting`
- Compile `g++ QuickSort.cpp ./lib/QuickSort.cpp` ，默认会输出 `a.out` 文件 (具体命令视情况而定)
- Run `./a.out`

### Fast Run
- Compile & Run `make run ./sorting/QuickSort.cpp`
